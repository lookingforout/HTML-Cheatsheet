HTML & CSS Cheatsheet

1. Основни структурни тагове

<html>

Определя началото и края на HTML документа.

<head>

Съдържа мета информация като заглавие, връзки към стилове и скриптове.

<title>

Определя заглавието на страницата, което се показва в раздела на браузъра.

<meta>

Задава мета данни за страницата, като кодиране и описания.

<link>

Връзка към външни ресурси, като CSS файлове.

<style>

Вгражда CSS код директно в HTML документа.

<script>

Добавя JavaScript код към страницата.

2. Тагове за оформление и секция

<body>

Съдържа основното съдържание на уеб страницата.

<header>

Определя горната част на страница или секция, често съдържа навигация или заглавие.

<footer>

Определя долната част на страница, съдържа информация за автора или линкове.

<section>

Представлява тематична област на съдържанието.

<article>

Определя самостоятелен блок от съдържание.

<nav>

Определя секция за навигационни връзки.

<aside>

Представлява допълнителна информация, като странична лента.

<main>

Определя основното съдържание на уеб страницата.

3. Тагове за заглавия и текст

<h1> - <h6>

Заглавни тагове, от най-голямо заглавие (<h1>) до най-малко (<h6>).

<p>

Създава параграфи от текст.

<a>

Определя хипервръзка към друга страница или ресурс.

<strong>

Подчертава важността на текста с удебеляване.

<em>

Добавя ударение върху текста с курсив.

<b>

Удебелява текста, без допълнителен смисъл за важност.

<i>

Прави текста курсив, без допълнителен смисъл за ударение.

<span>

Инлайн контейнер за текст или други елементи.

<br>

Добавя нов ред.

<hr>

Добавя хоризонтална линия за разделяне на съдържанието.

4. Тагове за списъци и медия

<ul>

Създава неподреден списък с маркери.

<ol>

Създава подреден списък който е нумериран.

<li>

Елемент от списък в <ul> или <ol>.

<img>

Вмъква изображение в страницата.

<audio>

Вгражда аудио плейър.

<video>

Вгражда видео плейър.

<iframe>

Вгражда друга уеб страница в рамка.

<figure>

Групира медийно съдържание като изображения или диаграми.

<figcaption>

Добавя надпис за съдържанието в <figure>.

5. Тагове за формуляри и таблици

<form>

Създава формуляр за въвеждане на данни от потребителя.

<input>

Представлява поле за въвеждане на данни.

<textarea>

Създава поле за многоредово въвеждане на текст.

<button>

Създава бутон за изпращане на данни или действие.

<select>

Създава падащ списък за избор.

<option>

Определя елемент от падащия списък <select>.

<label>

Свързва текст с форма или полета за въвеждане.

<table>

Създава таблица за организиране на данни в редове и колони.

<tr>

Определя ред в таблицата.

<td>

Създава клетка с данни в таблицата.

<th>

Създава клетка за заглавие в таблицата.

<thead>

Групира заглавните редове на таблицата.

<tbody>

Групира основните редове с данни в таблицата.

<tfoot>

Групира долния край на таблицата, съдържащ обобщение или бележки.

6. Тагове за взаимодействие и прогрес

<details>

Представя допълнителни подробности, които потребителят може да разгърне или свие.

<summary>

Заглавие за елемента <details>.

<progress>

Показва напредъка на задача.

<meter>

Представя измерване в определен диапазон, като процент.

1. CSS Свойства

Цветове и фонове

color: Определя цвета на текста.

background-color: Определя цвета на фона.

background-image: Настройва изображение за фон.

opacity: Регулира прозрачността на елемента.

Текст и шрифтове

font-family: Определя типа шрифт.

font-size: Задава размера на текста.

text-align: Определя подравняването на текста.

Бокс модел

margin: Определя външния отстъп около елемента.

padding: Определя вътрешния отстъп.

border: Определя рамка около елемента.

box-shadow: Добавя сянка на елемента.

Позициониране и оформление

display: Настройва стила на показване на елемента.

position: Определя позицията на елемента (relative, absolute, fixed).

flex: Позволява гъвкаво оформление.

Преходи и анимации

transition: Определя преход за стилове.

animation: Позволява анимации на елементи.

2. CSS Селектори

Универсален селектор

*: Избира всички елементи.

Селектор по тип

div, p, h1 и др: Избира елементи по техния тип.

Селектор по клас

.classname: Избира елементи с определен клас.

Селектор по ID

#idname: Избира елементи с определено ID.

Псевдо-класове

:hover, :active, :focus: Стилове при определени действия.

3. CSS шаблони с примери

Центриране на елемент

CSS код:


.centered {
    margin-left: auto;
    margin-right: auto;
    display: block;
    width: 50%;
}

Описание: Центрира елемент хоризонтално в контейнера му.

So centered

Сянка на елемент

CSS код:


.shadow {
    box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.3);
}

Описание: Добавя сянка с леко размазване.

So shadowy

Hover ефект над елемент

CSS код:


.hover-effect:hover {
    color: #B32113;
    transform: scale(1.05);
    transition: transform 0.3s ease;
}

Описание: Променя цвета и увеличава размера при hover.

Hover me

Респонсив грид оформление

CSS код:


.grid-layout {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
    gap: 15px;
}

Описание: Грид оформление, което се адаптира към различни размери на екрана.

Центриране с Flexbox

CSS код:


.flex-center {
    display: flex;
    align-items: center;
    justify-content: center;
    height: 100vh;
}

Описание: Центрира съдържанието хоризонтално и вертикално с Flexbox.

Flexbox centering

Закръглени ъгли

CSS код:


            .rounded-corners {
                border-radius: 15px;
                padding: 20px;
                background-color: #f0f0f0;
            }

Описание: Закръгля ъглите на елемента.

So roundy!

Сянка на текст

CSS код:


            .text-shadow {
                text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
                font-size: 1.2em;
            }

Описание: Добавя сянка на текста, за да изглежда изпъкващ.

Wow Shadows!

Бордър при hover

CSS код:


            .border-hover:hover {
                border: 2px solid #B32113;
                transition: border 0.3s ease;
            }

Описание: Показва бордър при hover върху елемента.

Бордър ефект при задържане

Фоново изображение на цял екран

CSS код:


            .full-width-bg {
                background-image: url('path/to/image.jpg');
                background-size: cover;
                background-position: center;
                height: 300px;
            }

Описание: Фоново изображение, което покрива цялата ширина на елемента.

Фоново изображение

Анимация за поява

CSS код:


            .fade-in {
                opacity: 0;
                animation: fadeIn 1.5s forwards;
                padding: 20px;
                background-color: #f0f0f0;
                text-align: center;
                margin: 0 auto;
                width: 50%;
            }
            
            @keyframes fadeIn {
                to {
                    opacity: 1;
                }
            }

Описание: Плавна поява на елемента.
